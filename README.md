이름 후보

HandyType
'Hand'와 'Type'을 결합한 이름으로, 손글씨(Handwriting)와 폰트(Typeface)의 의미를 담았습니다. 간결하면서도 직관적입니다.

MyScriptAI
'나만의 글씨(My Script)'와 'AI'를 합쳐, AI가 개인의 손글씨를 학습해 폰트로 만든다는 컨셉을 강조합니다.

PenGenius
'Pen(펜)'과 'Genius(천재)'를 합쳐, 손글씨를 천재적으로 변환해주는 AI의 이미지를 담았습니다.

FontCraft
'Font'과 'Craft(수공예)'를 결합해, AI가 손글씨를 예술적으로 폰트로 만든다는 느낌을 줍니다.

WriteAI
'Write(쓰다)'와 'AI'를 결합해, AI가 글을 쓰는 것처럼 손글씨 폰트를 생성한다는 의미를 담았습니다.

HandyFont
'Handy(손쉬운)'와 'Font'를 합쳐, 누구나 쉽게 자신의 손글씨 폰트를 만들 수 있다는 점을 강조합니다.

ScriptGen
'Script(글씨체)'와 'Generator'를 결합해, 손글씨 스크립트를 생성하는 AI의 역할을 잘 표현합니다.

InkAI
'Ink(잉크)'와 'AI'를 합쳐, 손글씨의 잉크 감성과 AI 기술을 동시에 담았습니다.

MyHandFont
'나만의 손글씨(My Handwriting)'와 'Font'를 결합해, 개인화된 폰트 생성을 강조합니다.

TypeMe
'Type'과 'Me'를 합쳐, '나만의 폰트를 만들어주는'이라는 의미를 담았습니다.
